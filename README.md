# AgentMail Skills (read-only mirror)

This repository is an auto-synced, read-only mirror of the `skills/` directory of
[agentmail-to/agentmail-plugins](https://github.com/agentmail-to/agentmail-plugins).

- **Do not open issues or PRs here** — contribute at
  [agentmail-plugins](https://github.com/agentmail-to/agentmail-plugins). Changes made
  directly to this repo are overwritten by the next sync.
- Install skills with `npx skills add agentmail-to/agentmail-skills` (unchanged), or
  install the full plugin: `/plugin install agentmail@agentmail` after
  `/plugin marketplace add agentmail-to/agentmail-plugins`.
- `agentmail-sdk` is a deprecated alias of `agentmail`, kept so existing
  `skills-lock.json` pins keep resolving.
